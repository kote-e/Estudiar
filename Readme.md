# Práctica HTML
Repositorio personal donde guardo ejercicios y prácticas de **HTML**.

## Descripción
Este repositorio contiene distintos archivos y ejemplos creados únicamente con fines de **aprendizaje y práctica**.  
No corresponde a proyectos finales ni profesionales, solo a pruebas y ejercicios mientras aprendo HTML.

## Contenido
- Ejercicios básicos de HTML  
- Pruebas de etiquetas y estructuras  
- Ejemplos simples de páginas web  
- Experimentos personales

> El contenido puede cambiar, estar incompleto o desordenado, ya que forma parte del proceso de aprendizaje.

## Objetivo
- Practicar HTML desde lo más básico  
- Entender la estructura de una página web  
- Mejorar progresivamente mis habilidades

## Uso
Repositorio de uso **personal**.  
No pensado para producción ni para ser utilizado como referencia profesional.
