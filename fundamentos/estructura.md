# Estructura básica de un documento HTML

A continuación se describen las etiquetas principales que forman la estructura básica de una página HTML.

---

## `<!DOCTYPE html>`
La declaración `<!DOCTYPE html>` define que este documento es un **documento HTML5**.  
Le indica al navegador qué tipo de HTML se está utilizando.

> No es una etiqueta, es una declaración.

---

## `<html>`
El elemento `<html>` es el **elemento raíz** de una página HTML.  
Todo el contenido del documento debe estar dentro de esta etiqueta.
Generalmente incluye el atributo `lang`, que indica el idioma del contenido.

Ejemplo:
```html
<html lang="es">´´´


---

## <head>
El elemento `<head>` contiene **metainformación** sobre la página HTML.  
Esta información no se muestra directamente en la página, pero es importante para el navegador.

Dentro de `<head>` se incluyen datos como:
- Título de la página
- Codificación de caracteres
- Configuraciones generales

---

## <title>
El elemento `<title>` especifica el **título de la página HTML**.  
Este título se muestra en la pestaña del navegador.

Ejemplo:
``` <title>Mi primera página</title> ´´´

---

## <body>
El elemento `<body>` define el **cuerpo del documento**.  
Contiene todo el contenido **visible** de la página web, como:
- Encabezados
- Párrafos
- Imágenes
- Enlaces
- Tablas
- Listas

---

## <h1>
El elemento `<h1>` define un **encabezado grande**.  
Se utiliza normalmente como el **título principal** de la página.

Se recomienda usar solo un `<h1>` por página.

---

## <p>
El elemento `<p>` define un **párrafo de texto**.  
Se utiliza para escribir texto normal dentro de una página web.

---

Estas etiquetas forman la base de cualquier documento HTML.
